Alhatorah Mobile Application

Transforming complex Jewish texts into an accessible mobile experience through systematic UX research and design.

Project in Development

Project Overview

A UX research and design initiative to transform Alhatorah's web platform into an intuitive mobile experience. Currently in design and prototyping phase.

2025 (In Progress)
Figma, Screaming Frog SEO Spider, Miro
UX Research & Design

Project Goals

  • Create intuitive mobile interface for Jewish text study
  • Simplify navigation between related texts
  • Optimize interactive learning tools
  • Enhance overall accessibility

Process & Development

Stage 1: Analysis

Content Classification

  • Static Resources: texts, commentaries
  • Dynamic Resources: tools, timelines

Key Findings

  • Inconsistent text displays
  • Complex navigation system
  • Cluttered interactive tools

Stage 2: Solutions

Design Approach

  • "Shtender" concept: focused single-resource view
  • Unified presentation system
  • Touch-optimized interactions

Key Features

Static Resources

  • Unified text display system
  • Collapsible commentaries
  • Streamlined profiles

Dynamic Tools

  • Horizontal timeline view
  • Touch-optimized lab interface
  • Enhanced mobile concordance

Current Progress

  • Completed initial platform analysis
  • Developed core design principles
  • Creating interactive prototypes
  • Preparing for user testing
