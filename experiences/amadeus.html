Company Background

Amadeus is a global leader in travel technology, providing cutting-edge solutions for the travel and tourism industry. With a presence in over 190 countries, Amadeus powers the global travel ecosystem, from search and booking to operations and customer experience.

I chose Amadeus because it represented the opportunity to work with highly skilled mentors while at the same time giving me the space to learn and make mistakes on my own. It was this guided independence that allowed to me learn and grow in a way that I would not have been able to in a traditional internship setting.

Main Project: PII Detection System

Project Goal

Develop an automated system to detect and flag Personally Identifiable Information (PII) in company metrics and documents, ensuring GDPR compliance and data privacy.

Technical Implementation

  • Architected a scalable Azure-based solution for processing large volumes of data
  • Implemented Tesseract OCR for scanning images and documents
  • Integrated Microsoft Presidio for enhanced PII detection
  • Created containerized microservices for modular functionality

Challenges & Solutions

  • Challenge: Processing diverse document formats
    Solution: Implemented multi-stage processing pipeline with format-specific handlers
  • Challenge: Accuracy of PII detection
    Solution: Combined multiple ML models and created custom rules for validation
  • Challenge: System scalability
    Solution: Utilized Azure's auto-scaling features and implemented efficient caching

Key Achievements

  • Achieved 95% accuracy using facial detection models
  • Achieved 90% accuracy using text detection models
  • Successfully integrated Microsoft Azure services into the system
